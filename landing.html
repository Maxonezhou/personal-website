Here is What I've Done so Far.

iPhone Hardware Intern @ Apple

Cupertino, CA

Performed circuit schematic capture and board design using Cadence software suite to support all aspects of iPhone product development. Worked with signal integrity team to help develop a suite of tools used to help simulate crosstalk, parasitic RLC components, perform S-parameter extractions, calculate transmission line parameters, and much more. Helped develope an oscilloscope automation process to capture I2C, I2S, SPI, and SPMI data directly from devices which sped up the validation process by several hours per interface.

Embedded Software Developer @ IGNIS Innovation

Waterloo, ON

Developed OLED display compensation algorithms and used the Windows Driver Model to implement algorithms into Windows software stack. Used OpenGL and Direct3D to modify pixel data for on the fly real-time rendering applications in OLED displays. Modified and built Android's SurfaceFlinger and Windows' WindowManager programs to implement display algorithms.

Backend Cloud Software Developer @ IBM 👁🐝𝗠

📍Ottawa, ON

I was a part the foundation team for IBM's Planning Analytics product, a cloud-based buisiness analytics tool running on the IBM TM1 multidimensional database used for enterprise scalable modeling. I worked on the backend cloud services helping build and integrate system-wide logging services using NodeJS, ExpressJS, Java/AspectJ, Apache HTTP Server, and Golang. Additionally, I worked on implementing a sanbox widget view refresh feature which allowed users to create multiple non-conflicting views of the same database while also reducing the total volume of database queries.

Hardware Virtual Forces Student Analyst @ CANSOFCOM

📍Ottawa, ON

I worked in the hardware division of the Canadian Special Operations Forces Command helping design and lead the deployment of new sensor hardware. I prototyped and leveraged IoT (Internet of Things) to integrate end-to-end solutions for sensor equipment and helped provide crucial information for command center personnel. Additionally, I used Flutter and Dart to create mobile apps to display sensor readings for users to view on the go.

Firmware Developer @ Ford Motor Company of Canada 🚙

📍Ottawa, ON

Our team worked on developing the firmware for the CAN gateway in future Ford vehicles. We helped fully integrate CAN software from database file configuration to full CAN routing support through all layers of the software stack. I worked on improving system parallelism and efficiency by implementing complete CPU, software stack, and runtime optimization using real-time Vector APIs and reducing ISR (interrupt service routine) overhead. I also developed an inter-process communication protocol using reason code.

Electrical Team - Sensor Interfacing @ WATOnomous 🚘

📍Waterloo, ON

Watonomous is the University of Waterloo's student run autonomous vehicles team competing in the SAE Autodrive Challenge. Our team is made up of four main divisions (Software, Electrical, Mechanical, and Buisness) as well as several subteams working towards turning a Chevorlet Bolt into a fully capable autonomous vehicle. I work on the sensor interfacing team where I write driver code for our vehicle's sensors such as LiDARs, cameras, and GPS. This semester I worked on integrating TF transforms to unify the frames of reference for the sensors.

Fluid Dynamics Student Researcher @ University of Calgary 🔬

📍Calgary, AB

Worked as a fluid dynamics student researcher at the University of Calgary over the summers of 2017 and 2018 where I carried out multi-input and multi- output (MIMO) system identification on fluid dynamic systems using Matlab and Simulink to calculate fluid flow parameters. I also experimentally implemented Buttersworth filters to reduce statistical noise by 77%.
